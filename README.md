# hot-pies
A shoot 'em up adventure involving farmers, cows and UFOs.

### Steps
1. 'npm install'
2. 'npm start'
3. Access 'http://localhost:8000/test' to run tests.
4. Access 'http://localhost:8000/' to play the game.
